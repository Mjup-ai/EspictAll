## 🎯 事業概要（クイックサマリー）

### 新事業の核心

株式会社Mjupは、既存事業「エスピクトスクール」に加え、新たに「エスピクトワーク」事業を立ち上げ、**障害者雇用支援の総合ソリューション企業**へと進化します。

### ビジネスモデル

```
┌─────────────────────────────────────────────────┐
│          エスピクトスクール（既存事業）            │
│       就労移行支援事業所向けシステム提供          │
│         障害者の就労スキル習得支援               │
└────────────────┬────────────────────────┘
                     │ 習得者を排出
                     │ データ連携
                     ▼
┌─────────────────────────────────────────────────┐
│          エスピクトワーク（新規事業）             │
│         ・障害者雇用斡旋サービス                  │
│         ・障害者雇用管理代行サービス              │
│         ・企業での定着支援                       │
└─────────────────────────────────────────────────┘
```

## 💡 差別化ポイント

### 競合（エスプール）との違い

| 比較項目 | 競合（エスプール） | Mjup |
|---------|-------------------|------|
| 業務内容 | 農業 | IT・事務スキル |
| 人材育成 | なし | エスピクトスクール連携 |
| データ活用 | 限定的 | AI・システム統合 |
| 成長機会 | 少ない | キャリアパス提供 |
| 対象企業 | 大手企業 | 中堅～大手 |

### 独自の強み

1. **垂直統合モデル**: 育成→就職→定着までワンストップ
2. **データドリブン**: 習得データに基づく最適マッチング
3. **継続成長支援**: 配置後もスキルアップを継続
4. **IT特化**: 農業ではないIT・事務職での活躍
5. **システム連携**: エスピクトスクールとのシームレス連携

## 🎯 ビジョン

### 5年後の姿

```
「障害者雇用といえばMjup」

エスプールが農業×障害者雇用なら、
MjupはIT×障害者雇用のパイオニアへ。

ただ雇用するだけでなく、
一人ひとりの成長を支援し、
データで証明する。

それがMjupの新しい挑戦。
```

### 最終的なゴール

- **障害者**: 安定した雇用と成長機会の獲得
- **企業**: 法令遵守と優秀な人材の確保
- **Mjup**: 持続可能な事業と社会貢献の両立

**三方良しのビジネスモデル**
