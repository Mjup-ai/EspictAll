✓ 生産活動の収益は利用者に工賃として支払う

### 3. 5つの核心メリット（5スライド or 1スライド）

**各メリットを以下の構造で統一：**

#### ① 工賃を生む
```
【実務案件が含まれるプラン設計】

エントリープラン（30万円）: 年間10万円分
スタンダードプラン（50万円）: 年間30万円分
プレミアムプラン（100万円）: 年間60万円分

効果：
• 毎月一定量の生産活動を供給
• 工賃が安定 → 利用者定着
• 工賃向上計画が実務と連動

「工賃を"頑張る"から"回る"に変える」
```

**視覚化の指示：**
- プラン別の金額をグラフや表で表示
- 「頑張る→回る」を矢印で表現

#### ② 指導員を活かす
```
【目標工賃達成指導員配置を「形だけ」で終わらせない】

✓ 計画→案件→実績→評価が一本で回る
✓ 指導員が実際に機能する環境を提供
✓ 不足分（0.2〜0.4）だけの補填で成立する可能性が高い

効果：
• 指導員配置を「名ばかり」にしない
• 加算を"取りにいく"から"取り続ける"へ

「人を置くためのサービスではなく、
置いた人が機能する環境を作るサービス」
```

**視覚化の指示：**
- 計画→案件→実績→評価のフローを図解
- 循環する矢印で表現

#### ③ 加算を安定させる
```
【説明可能な工賃原資の提供】

✓ 第三者取引で説明可能な工賃原資に寄せられる
✓ 実務案件の実績データが残る
✓ 監査・実地指導に備えられる

効果：
• 加算取得の根拠が明確
• 将来改定・実地指導リスクの軽減
• 継続的な加算取得が可能
```

**視覚化の指示：**
- 「説明可能」を強調（キーワード）
- データが残ることを図解

#### ④ 属人化を外す
```
【生産活動・教育・案件供給を仕組み化】

✓ システム化されたトレーニングコンテンツ
✓ 標準化された実務案件の提供
✓ 運用マニュアルとサポート体制

効果：
• 特定職員・管理者への依存を削減
• 職員疲弊・離職リスクの軽減
• 今の安定を"再現可能な形"に変える
```

**視覚化の指示：**
- 「人頼み」と「仕組み」を対比図で表現
- 属人化→仕組み化の変化を矢印で

#### ⑤ 監査・将来改定に備える
```
【データに基づく説明力】

✓ スキル習得データの蓄積
✓ 実務案件の実績データ
✓ 工賃向上計画との連動

効果：
• 原資説明が明確
• 実地指導での説明力
• 将来改定への対応力
```

**視覚化の指示：**
- データの蓄積を図解（データベースイメージ）
- 説明力の向上をグラフで表現

## 🏢 事業所タイプ別メリット（4スライド）

### 各タイプを以下の構造で統一：

#### ① 利用者数に困っていない × 生産活動にも困っていない事業所
```
【表の顔】
✓ 定員稼働
✓ 工賃も出ている
✓ 生産活動も安定

【裏のリスク】
⚠ 属人化（特定職員・管理者依存）
⚠ 原資説明が弱い／慣習運用
⚠ 将来改定・実地指導リスク
⚠ 職員疲弊・離職リスク

【エスピクトの価値】
• 生産活動・教育・案件供給を仕組み化
• 第三者取引で説明可能な工賃原資に寄せられる
• 目標工賃達成指導員が「名ばかり」にならない
• 今の安定を"再現可能な形"に変える

「今うまくいっている状態を、将来も続けるための保険」
```

**視覚化の指示：**
- 表の顔と裏のリスクを左右に配置
- 「保険」を強調

#### ② 利用者数は安定 × 生産活動が不安定な事業所
```
【課題】
⚠ 仕事が途切れる
⚠ 工賃が上下する
⚠ 職員が仕事探しに追われる

【エスピクトの価値】
• 毎月一定量の生産活動を供給（年間10万円〜60万円分）
• 工賃が安定 → 利用者定着
• 工賃向上計画が実務と連動

「工賃を"頑張る"から"回る"に変える」
```

**視覚化の指示：**
- 不安定なグラフ→安定したグラフの変化を図解
- 「頑張る→回る」を強調

#### ③ 利用者数が不安定 × 生産活動はある事業所
```
【課題】
⚠ 差別化が弱い
⚠ 新規・紹介に刺さらない

【エスピクトの価値】
• IT×実務×生産活動という明確な売り
• 在宅支援・若年層対応
• 見学時の説明が強くなる

「選ばれる理由を一気に作れる」
```

**視覚化の指示：**
- IT×実務×生産活動の3つの要素を図解
- 「選ばれる理由」を強調

#### ④ 利用者数も生産活動も不安定な事業所
```
【課題】
⚠ 工賃が出ない

⚠ 将来不安が強い

【エスピクトの価値】
• 生産活動の土台を外付け（年間10万円〜60万円分）
• 職員教育・業務負担を軽減
• 立て直しの"最短ルート"

「ゼロから作らず、外付けで立て直す」
```

**視覚化の指示：**
- 「ゼロから作る」と「外付け」を対比図で表現

## 📊 プラン別メリット（3スライド）

### 各プランを以下の構造で統一：

#### エントリープラン（30万円）
```
【導入しやすい入口】

年間料金: 30万円
仕事依頼額: 10万円/年
「30万で10万の実務課題」だと数字の納得感が上がる

【含まれる内容】
✓ 基礎コース3コース（タイピング、データ入力、ビジネスマナー）
✓ 実務案件：年間10万円分（簡単なデータ入力・短時間文字起こし）
✓ 月1回の品質チェック・運用レビュー
✓ 基本マニュアル提供

【こんな事業所におすすめ】
• 初めてIT教育を導入する事業所
• 小規模事業所（利用者10名程度）
• まずは試してみたい事業所
```

**視覚化の指示：**
- 料金と仕事依頼額を大きく表示
- 「30万で10万」を強調

#### スタンダードプラン（50万円）
```
【実務に直結する、本格的なスキル習得】

年間料金: 50万円
仕事依頼額: 30万円/年

【含まれる内容】
✓ 全コースアクセス（基礎3＋応用5＋専門7コース）
✓ 実務案件：年間30万円分（多様な案件・中級レベル）
✓ 月2回の品質チェック・運用レビュー（60分）
✓ 詳細マニュアル + カスタムマニュアル

【30万円プランとの違い】
• 学べる範囲と実務の深さが変わる
• 添削・検品の頻度が2倍
• 運用設計のサポートが手厚い

【こんな事業所におすすめ】
• 本格的にITスキル教育を導入したい事業所
• 中規模事業所（利用者30名程度）
• 工賃向上を目指す事業所
```

**視覚化の指示：**
- 30万円との比較表を追加
- 「2倍」を強調

#### プレミアムプラン（100万円）
```
【大規模運用・複数チーム対応】

年間料金: 100万円
仕事依頼額: 60万円/年（スタンダードの2倍）

【含まれる内容】
✓ 全コースアクセス（スタンダードと同じ）
✓ 実務案件：年間60万円分（スタンダードの2倍）
✓ 月4回の品質チェック・運用レビュー（90分、2倍）
✓ 2ライン運用対応（複数チーム運用可）
✓ 担当者向けの運用研修（職員側）付き

【50万円プランとの違い】
• カリキュラム内容は同じ
• 実務案件の量が2倍
• 運用密度が2倍（品質チェック・レビュー）
• 複数チーム運用が可能

【「50万×2」ではなく、1つのプランとして運用するメリット】
• 統一された品質管理
• 効率的な運用（1つのプランで複数チーム管理）
• 専任サポートによる手厚い支援

【こんな事業所におすすめ】
• 大規模事業所（利用者50名以上）
• 複数チームで運用したい事業所
• 本格的な工賃向上を目指す事業所
• 高い品質管理を求める事業所
```

**視覚化の指示：**
- 50万円との比較表を追加
- 「2倍」を強調
- 複数チーム運用を図解

## 🎨 デザイン・レイアウトの指示

### 全体
- **カラースキーム**: プロフェッショナル（青・緑・金をアクセント）
- **フォント**: 読みやすいサンセリフ体
- **余白**: 十分な余白を確保

### 強調ポイント
- **数字**: 大きく、太字で表示
- **「2倍」「保険」「回る」などのキーワード**: 色を変えて強調
- **矢印**: 変化や流れを明確に

### 図解
- **フロー図**: 計画→案件→実績→評価
- **比較図**: プラン間の違い
- **対比図**: 人頼み vs 仕組み、頑張る vs 回る
